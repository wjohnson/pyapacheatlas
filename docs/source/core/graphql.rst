=================
GraphQL Utilities
=================
.. currentmodule:: pyapacheatlas.core.graphql

.. autosummary::
   :toctree: api/

   GraphQLClient
   GraphQLException

The GraphQLClient provides utility functions to the GraphQL SearchAPI.
Specifically, it includes the `query` method to execute a GraphQL query
across the Microsoft Purview Data Map objects.

See the official Microsoft Purview Docs for query examples: https://learn.microsoft.com/en-us/purview/tutorial-graphql-api
